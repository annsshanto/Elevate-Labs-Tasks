TASK 1

 

1. Install Nmap from official website

I have already downloaded Nmap and to know nmap version type this command.

nmap --version

 

2. Find your local IP range.

To know the local IP use this command

Ifconfig

 

3. Run: nmap -sS 192.168.1.0/24 to perform TCP SYN scan.

 

 

5. Optionally analyze packet capture with Wireshark

 

5,6. Research common services running on those ports.

·       SSH   -22

·       FTP   -21

·       telnet   -23

·       domain  -53

·       http  -80

·       https  -443

 

7. Identify potential security risks from open ports.

Security Risks of SSH

  • Brute-force password attacks
  • Credential stuffing
  • Weak authentication

 

Security Risks of FTP

·        Credential sniffing

·        Anonymous login abuse

·        File manipulation

 

Security Risks of Telnet

·        Credentials visible in plaintext

·        Session hijacking

·        Man-in-the-middle attacks

 

Security Risks of Domain

·        DNS amplification attacks

·        Cache poisoning

·        Open DNS resolver misuse

 

Security Risks of HTTP

·        Credential theft

·        Session hijacking

·        Content manipulation

 

Security Risks of HTTPS

·        Weak TLS versions

·        Expired certificates

·        Vulnerable web applications
